# StudyHub

A full-featured study management web app built with **Python Flask** + **SQLite**.

## Features
- Dashboard with today's timetable
- Color-coded Subjects
- Weekly Timetable
- Study Plans with progress tracking
- Upload/Download PDF & DOCX materials
- Study session logging & statistics
